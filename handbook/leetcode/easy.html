    # 两数之和（35）

    # 题目

    给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

    # 例子

    输入：nums = [2,7,11,15], target = 9

    输出：[0,1]

    解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。

    # 思路

    暴力枚举法，双层for循环，第一层从0开始，第二层从第一层加1开始，当找到时return即可

    # 代码

    var twoSum = function(nums, target) {
    for(let i = 0;i<nums.length;i++){
        for(let j=i+1;j<nums.length;j++){
            if(nums[i]+nums[j] === target){
                return [i,j]
            }
        }
    }
};

    # 回文数

    # 题目

    给你一个整数 x ，如果 x 是一个回文整数，返回 true ；否则，返回 false 。 回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

    # 例子

    输入：x = 121

    输出：true

    # 思路

    将整数x转换为字符串，对其使用字符串方法，当其长度为2的整数倍时，使用slice(0,x.length/2)(length/2,length)对比， 当其长度不为2的整数倍时，使用slice(0,x.length/2)(x.length/2+1,x.length)对比。 这里的一个问题是后面截取出来的字符串需要翻转，比如1001，截取后是10、01，后面的翻转再对比即可

    # 代码

    var isPalindrome = function(x) {
    const xx = x.toString()
    if(xx.length%2 === 0){
        const xxx = xx.slice(0,xx.length/2)
        const xxxx = xx.slice(xx.length/2,xx.length).split('').reverse().join('')
        return xxx === xxxx;
    }else{
        const xxx = xx.slice(0,xx.length/2)
        const xxxx = xx.slice(xx.length/2+1,xx.length).split('').reverse().join('')
        return xxx === xxxx
    }
};

    # 最长公共前缀（7）

    # 题目

    编写一个函数来查找字符串数组中的最长公共前缀。 如果不存在公共前缀，返回空字符串 ""。

    # 例子

    输入：strs = ["flower","flow","flight"]

    输出："fl"

    输入：strs = ["dog","racecar","car"]

    输出：""

    解释：输入不存在公共前缀。

    # 思路

    使用双层循环进行判断，第一层代表传入的第一个元素的每一个字符，第二层代表传入的第几个元素，使用strs[i][j]表示第i个元素的第j个字符， 全程只需要使用数组第0项和其他项对比即可，如果每一项从0开始到第j个字符都相等，那它就是一个公共前缀，放在re存储起来，一旦在对比过程中 发现某一个字符和第0项的不相等，就直接返回re。

    # 代码

    var longestCommonPrefix = function(strs) {
    let re = ''
    if(strs[0].length === 0){ return re}
    if(strs.length === 1){ return strs[0]}
    for(let j = 0;j< strs[0].length;j++){
        for(let i = 1;i<strs.length;i++){
            if(strs[0][j] !== strs[i][j]){return re}
        }
        re+=strs[0][j]
    }
    return re
};

    # 有效的括号（37）

    # 题目

    给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。

    # 例子

    输入：输入：s = "()[]{}"

    输出：true

    输入：s = "([)]"

    输出：false

    # 思路

    使用入栈和出栈的思路，遇到一个左括号时，将其放入栈中，当遇到一个右括号时，从栈中拿到最后一次入栈的元素，这个元素应该是和当前 这个右括号对应的左括号，如果不能对应直接返回false。如果所有的元素都能对应上，那么最后的栈应该是空数组，如果最后栈中还存在元素， 那就表明左括号多出，同样返回false。

    # 代码

    var isValid = function(s) {
    let arr = [];
    let mp = new Map()
    mp['('] = ')'
    mp['['] = ']'
    mp['{'] = '}'
    for(c of s){
        if(c in mp){
            arr.push(c)
        }else{
            const top = arr.pop()
            if(c !== mp[top]){return false}
        }
    }
    return !arr.length;
};

    # 合并两个有序链表（14）

    # 题目

    将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

    # 例子

    输入：l1 = [1,2,4], l2 = [1,3,4]

    输出：[1,1,2,3,4,4]

    输入：l1 = [], l2 = []

    输出：[]

    # 思路

    首先需要理解链表是什么样的结构，说白了其实就是一个数组，只是可以使用val和next来表示当前值和下一个值。 而本题的思路可以是这样的：当l1和l2都不是空的时候，判断l1和l2哪一个链表的头节点值更小，将较小的值节点 添加到结果，同时将对应链表的节点向后移一位。 我们设置一个空链表，我们想要做的就是将这个链表的next指向l1和l2中当前节点对比后较小的那一个，然后将对应 链表指向他的next，我们不断重复这个过程，直到其中一个链表为空，再将剩余的那个拼在我们得到的链表后面即可。

    # 代码

    var mergeTwoLists = function (list1, list2) {
    function ListNode(val, next) {
        this.val = (val === undefined ? 0 : val)
        this.next = (next === undefined ? null : next)
    }
    const prehead = new ListNode(-1);
    let prev = prehead;
    while (list1 !== null && list2 !== null) {
        if (list1.val <= list2.val) {
            prev.next = list1;
            list1 = list1.next
        } else {
            prev.next = list2;
            list2 = list2.next
        }
        prev = prev.next
    }

    prev.next = list1 === null ? list2 : list1;
    return prehead.next
};

    # 删除有序数组中的重复项（2）

    # 题目

    给你一个 升序排列 的数组 nums ，请你 原地 删除重复出现的元素，使每个元素 只出现一次 ，返回删除后数组的新长度。元素的 相对顺序 应该保持 一致 你必须在 原地 修改输入数组 并在使用 O(1) 额外空间的条件下完成。

    # 例子

    输入：nums = [0,0,1,1,1,2,2,3,3,4]

    输出：5, nums = [0,1,2,3,4]

    解释：函数应该返回新的长度 5 ， 并且原数组 nums 的前五个元素被修改为 0, 1, 2, 3, 4 。不需要考虑数组中超出新长度后面的元素。

    # 思路

    本题可以使用双指针解决。首先由于是一个有序数组，因此我们在对比的时候就会比较简单，我们设置一个fast指针，一个slow指针，两者同时从 1开始算起，fast指针先走，走的同时对比nums[fast]和nums[fast-1]，当前后值相同时说明有一段重复的值，一直走到两个值不同，说明这一段 相同的值结束，开始下一段相同的值，这时需要将nums[slow]的值更新为nums[fast]的值，再将slow前进一位，等待接收下一个不同的值，直到fast 将nums走完。这时nums里从0到slow就是我们要的不重复的数组。

    # 代码

    var removeDuplicates = function(nums) {
    if(nums.length === 0){return 0}
    let fast = 1,slow = 1
    for(fast;fast<nums.length;fast++){
        if(nums[fast] !== nums[fast-1]){
            nums[slow] = nums[fast]
            slow++
        }
    }
    return nums.slice(0,slow)
};

    # 爬楼梯（26）

    # 题目

    假设你正在爬楼梯。需要 n 阶你才能到达楼顶。 每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

    # 例子

    输入：n = 3

    输出：3

    解释：有三种方法可以爬到楼顶。

    # 思路

    本题可以使用双指针解决。首先由于是一个有序数组，因此我们在对比的时候就会比较简单，我们设置一个fast指针，一个slow指针，两者同时从 1开始算起，fast指针先走，走的同时对比nums[fast]和nums[fast-1]，当前后值相同时说明有一段重复的值，一直走到两个值不同，说明这一段 相同的值结束，开始下一段相同的值，这时需要将nums[slow]的值更新为nums[fast]的值，再将slow前进一位，等待接收下一个不同的值，直到fast 将nums走完。这时nums里从0到slow就是我们要的不重复的数组。

    # 代码

    var climbStairs = function(n) {
    let p = 0,q = 0,r=1
    for(let i = 0;i<n;i++){
        p = q
        q = r
        r = p+q
    }
    return r
};

    # 移除元素

    # 题目

    给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。

    # 例子

    输入：nums = [0,1,2,2,3,0,4,2], val = 2

    输出：5, nums = [0,1,4,0,3]

    解释：函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。注意这五个元素可为任意顺序。你不需要考虑数组中超出新长度后面的元素。

    # 思路

    本题可以使用双指针解决。我们设置左指针从第一项开始，右指针从最后一项开始。通过左指针与val对比，如果不相等说明没有问题， left++指向下一个对比位。 如果相等就将右指针指向的值赋给左边，同时right--，如果右指针赋过来的值还是等于val，那么还是 一样的情况，将右指针指向的值赋给左边，同时right--，直到left等于right时返回left的值就是答案。

    # 代码

    var removeElement = function (nums, val) {
    let left = 0;
    let right = nums.length
    while (left < right) {
        if (nums[left] === val) {
            nums[left] = nums[right-1]
            right--
        } else {
            left++
        }
    }
    return left
};

    # 二分查找

    # 题目

    给定一个 n 个元素有序的（升序）整型数组 nums 和一个目标值 target ，写一个函数搜索 nums 中的 target，如果目标值存在返回下标，否则返回 -1。

    # 例子

    输入：nums = [-1,0,3,5,9,12], target = 9

    输出：4

    解释：9 出现在 nums 中并且下标为 4

    # 思路

    首先这里为升序排列，因此可以使用二分查找。我们定义左右两个指针，l=0，r=nums.length-1，分别指向数组的第一个和最后一个，然后取数组的中间和target对比， 如果相等则直接返回下标即可。如果数组中间数大于target，则说明target只可能在nums前半段（不包括中间数），所以将right设置为中间的index-1，left不变， 如果数组中间数小于target，则说明target只可能在nums后半段（不包括中间数），所以将left设置为中间的index+1，right不变

    # 代码

    var search = function(nums, target) {
    let left = 0, right = nums.length-1
    while(left<=right){
        let mid = parseInt((left+right) / 2)
        if(nums[mid] === target){
            return mid
        }
        let isSmall = nums[mid] < target
        left = isSmall ? mid + 1: left;
        right = isSmall ? right : mid - 1
    }
    return -1
};
